# Flux Open Home — Home Assistant Add-ons

Custom Home Assistant add-on repository for Flux Open Home products.

## Add-ons

### Flux Irrigation Management API
A secure, scoped REST API that lets irrigation management companies monitor and control your Flux Open Home irrigation system — without accessing any other devices in your Home Assistant.

## Installation

1. Open Home Assistant
2. Go to **Settings → Add-ons → Add-on Store**
3. Click the **⋮** (three dots) menu in the top right
4. Click **Repositories**
5. Paste this URL: `https://github.com/FluxOpenHome/ha-flux-irrigation-addons`
6. Click **Add → Close**
7. The Flux Irrigation Management API will appear in the add-on store
